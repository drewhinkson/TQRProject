# TQRProject

## Overview
TQRProject is a Node.js application designed to integrate with Slack and Zendesk. It provides functionalities for handling Slack interactions, posting modals, processing modal submissions, and updating Zendesk tickets based on the user's input.

## Features
- **Slack Integration**: Interact with Slack to receive and handle user inputs.
- **Modal Posting**: Post interactive modals to Slack for user inputs.
- **Zendesk Integration**: Update Zendesk tickets based on the information received from Slack modals.

## Installation
To set up the TQRProject, follow these steps:
1. Clone the repository:
2. Run NPM install 


## Configuration
Before running the application, configure the necessary environment variables:
- `SLACK_BOT_TOKEN`: Your Slack bot token.
- `ZENDESK_EMAIL`: Your Zendesk email.
- `ZENDESK_API_TOKEN`: Your Zendesk API token.
- `ZENDESK_SUBDOMAIN`: Your Zendesk subdomain.
